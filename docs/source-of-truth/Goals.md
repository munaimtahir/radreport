# Goals.md

## MVP (v1) Goals
1. Reception can:
   - create/find patient
   - generate MR
   - create visit (SV)
   - add services from catalog
   - apply discount + accept partial payments
   - compute due
   - print A4 duplicate receipt (patient + office copy)

2. Worklists:
   - OPD worklist shows visits/services that include OPD
   - USG worklist shows visits/services that include ultrasound

3. Ultrasound:
   - data operator enters findings
   - verifier can return / verify
   - final report PDF generated and printable

4. OPD:
   - operator enters vitals
   - doctor enters prescription
   - final prescription PDF generated and printable

5. Admin (minimum):
   - manage Service Catalog (rates, active)
   - manage users/roles
   - view audit log

## v1.0 (Soon After MVP)
