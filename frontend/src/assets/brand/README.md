# Brand Assets

## Logo Files Required

Please add the following logo files to this directory:

1. **Consultants_Place_Clinic_Logo.png** - Full logo with text on white background
2. **Consultants_Place_Clinic_Logo_Transparent.png** - Logo with transparent background (preferred)

## Brand Colors

Extracted from the logo:

- **Brand Blue**: `#0B5ED7` - Used for primary actions, links, active states
- **Brand Orange**: `#FF6B35` - Used for accents, secondary actions, highlights

## Usage

Import logo in components:
```typescript
import logo from '@/assets/brand/Consultants_Place_Clinic_Logo_Transparent.png';
```

The logo displays:
- A stylized 'c' in blue + 'p' in orange at the top
- Text below: "Consultants • Place • Clinic" (blue • orange • orange)
